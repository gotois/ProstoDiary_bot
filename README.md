Install
---
```bash
npm rebuild
npm i
chmod +x bin/index 
```

Package upgrade
---
```bash
npm run package-upgrade
```

Usage
---
Checking Unit tests:
```bash
npm run unit
```

Run development:
```bash
npm run dev
```

Run production:
```bash
npm start
```

Global Bot Environments
---
```
Name: ProstoDiary
Username: @ProstoDiary_bot
```

Run Telegram Bot
```
/start
```

docs
---
```
cd docs
bundle exec jekyll serve
```
