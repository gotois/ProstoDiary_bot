Install
---
```bash
npm rebuild
npm i
```

```bash
chmod +x bin/index 
```


Run Telegram Bot
```
/start
```

After change node version
```bash
npm rebuild
```

Usage
---
Checking Unit tests:
```bash
npm run test:watch
```

Run development:
```bash
npm run dev
```

Run production:
```bash
npm start
```
